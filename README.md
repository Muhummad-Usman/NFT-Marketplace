# NFT Trading Platform

A decentralized NFT marketplace built with Solidity and Web3 that allows users to mint, list, and trade NFTs securely on the blockchain. The platform supports multiple NFT standards and integrates decentralized storage for digital assets.

## 🚀 Features

- Multi-standard NFT marketplace
- Supports **ERC-721**, **ERC-1155**, and **ERC-2981 royalty standard**
- Mint NFTs including **images, audio, and video**
- List and trade NFTs in a decentralized environment
- Royalty support for creators
- Web3-enabled frontend for seamless blockchain interaction
- Integration with decentralized storage

## 🛠️ Tech Stack

- **Solidity** – Smart contract development  
- **Web3.js / Ethers.js** – Blockchain interaction  
- **Hardhat** – Development and testing environment  
- **Ethereum** – Blockchain network  
- **IPFS / Decentralized Storage** – Asset storage
